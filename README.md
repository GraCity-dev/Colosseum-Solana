# 🌿 **Colosseum-Solana**  
### _GraCity Impact DApp – Building Regenerative Cities on Solana_

> 🪙 Official Submission for the **Colosseum Solana Hackathon 2025**  
> 🌍 A Web3 solution to measure, verify, and finance real-world urban sustainability impact.

---

## 💡 **Project Vision**

**GraCity Impact DApp** is an **AI + Blockchain** platform that turns **urban sustainability projects** into **traceable, investable impact assets**.  
It bridges **IoT environmental data**, **ESG analytics**, and **tokenized proofs** — empowering transparent, regenerative cities.

We believe **impact data should be public, verifiable, and rewarded.**

---

## 🔎 **Core Features**

✅ **Impact Heatmap** – Real-time visualization of ESG metrics (energy, air quality, green coverage).  
✅ **Tokenized Investments** – Fractional ownership of certified green projects.  
✅ **AI + ESG Analytics** – Predictive insights on sustainability performance.  
✅ **NFT Certificates** – On-chain verification of environmental impact.  
✅ **Open ReFi Infrastructure** – Connects cities, investors, and citizens under transparent governance.  

---

## ⚙️ **Tech Stack**

| Layer | Tools & Frameworks |
|:------|:--------------------|
| **Blockchain** | Solana (Testnet), Anchor Framework |
| **Frontend** | React + Vite + TailwindCSS + TypeScript |
| **Wallet Integration** | Phantom / Solflare via `@solana/wallet-adapter` |
| **NFT & Metadata** | Metaplex SDK |
| **Data Layer** | IoT APIs, Pyth Network Oracle |
| **Hosting** | Netlify / Vercel (for demo) |
